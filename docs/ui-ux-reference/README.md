# UI/UX Design Reference — Página Matemáticas

Repositorio de referencia para principios modernos de diseño UI/UX.
Úsalo como guía cuando construyas o mejores interfaces con agentes IA.

## Archivos

| Archivo | Contenido |
|---------|-----------|
| [`DESIGN.md`](./DESIGN.md) | Sistema completo: tokens de color, tipografía, espaciado, animaciones, accesibilidad |
| [`COMPONENTS.md`](./COMPONENTS.md) | Patrones HTML/CSS: navbar, hero, bento grid, cards, formularios, modales, skeletons |

## Cómo usar con Claude

Cuando quieras construir una nueva sección o componente, indica a Claude:

```
Usa los principios de /docs/ui-ux-reference/DESIGN.md para construir [componente].
```

O invoca la skill directamente:

```
/modern-ui-ux
```
